# 基于springboot的医院管理系统4

#### Description
基于springboot的医院管理系统拥有四种角色

- 管理员：医生管理、账号管理、病患管理、报表统计等
- 药品管理员：药品管理、药房结算等
- 医生：查看药品、查看病患信息、患者药方管理
- 患者：查看药方
