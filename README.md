# 基于springboot的医院管理系统



## 1、项目介绍

基于springboot的医院管理系统拥有四种角色

- 管理员：医生管理、账号管理、病患管理、报表统计等
- 药品管理员：药品管理、药房结算等
- 医生：查看药品、查看病患信息、患者药方管理
- 患者：查看药方


## 2、项目技术

后端框架：springboot

前端技术：Layui、html、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- tomcat版本：不需要
- 数据库类型：MySql（5.x和8.x版本都可） 
- maven版本：无限制
- 硬件环境：Windows 或者 Mac OS
